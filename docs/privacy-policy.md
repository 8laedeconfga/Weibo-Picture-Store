# Privacy Policy

目前，我们没有收集任何用户信息。

## Use of Permissions

| 权限                          | 用途说明                           |
| ----------------------------- | ---------------------------------- |
| clipboardWrite                | 把上传后获得的图片 URL 写入剪切板  |
| clipboardRead                 | 从剪切板读取要上传的图片 URL       |
| downloads                     | 导出日志                           |
| contextMenus                  | 网页中捕获图片、视频的右键菜单入口 |
| notifications                 | 错误信息提示                       |
| storage                       | 存储配置信息                       |
| webRequest/webRequestBlocking | 修改微博相关 APIs                  |
| tabs/\<all_urls\>             | 捕获任意页面内的图片和视频         |
